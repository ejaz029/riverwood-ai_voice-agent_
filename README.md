# AI Voice Agent 🎙️

An AI-powered multilingual voice assistant

## 🚀 Features

- Voice-based conversational AI
- Text-to-Speech (TTS)
- Context-aware memory
- Hindi + English interaction
- LLM-powered intelligent replies
- Construction-domain conversational prompts

## 📂 Project Structure

```bash
project/
├── main.py
├── config.py
├── llm.py
├── tts.py
├── input_io.py
├── memory.py
├── requirements.txt
└── README.md
🛠️ Installation
git clone https://github.com/yourusername/riverwood-ai-agent.git
cd riverwood-ai-agent
pip install -r requirements.txt
🔐 Environment Variables

Create a .env file:

OPENAI_API_KEY=your_api_key
RIVERWOOD_MAX_TURNS=6
RIVERWOOD_MAX_SECONDS=120
▶️ Run the Project
python main.py
💬 Example
Miss Riverwood:
Namaste sir, chai pee li?
Main Miss Riverwood hoon.

You:
Haan, contractor meeting hui thi.

Miss Riverwood:
Achha sir, budget discussion kaisa raha?
